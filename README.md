# Life Hack or Urban Myth?

A simple Android flashcard game built using Kotlin to help users distinguish between real life hacks and viral urban myths.

## Overview
This application presents users with a series of common "hacks." Users must decide if each statement is a real "Hack" (True) or an "Urban Myth" (False). The app tracks the user's score and provides detailed explanations for each answer.

## Features
- Interactive Quiz: Test your common sense with 5 different life hacks.
- Instant Feedback: Learn why a hack is real or fake immediately after answering.
- Score Tracking: View your final results at the end of the session.
- Review Mode: Re-examine all questions and their detailed explanations.
- Simplified Navigation: Uses a single-activity layout for a smooth, fast user experience.

## Technical Details
- Language: Kotlin
- Logic: parallel arrays for data storage and view visibility toggling for screen navigation.
- CI/CD: Integrated with GitHub Actions to automate the build process and ensure code quality.

## How to Run
1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the app on an emulator.
